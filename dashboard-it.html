🔗 COMMENT CONNECTER LE DASHBOARD
1
Ouvrir le fichier HTML
Télécharge ce fichier sur ton PC. Double-clique dessus → il s'ouvre dans Chrome ou Edge.
2
Partager tes Google Sheets
Dans chaque Sheet → Partager → Tout le monde avec le lien → rôle Lecteur. Copie l'ID dans l'URL entre /d/ et /edit.
3
Coller les IDs ci-dessous
Remplis les champs dans le panneau de connexion avec les IDs de tes Sheets. Clique sur Connecter.
4
Synchronisation auto
Le dashboard se met à jour automatiquement. Tu peux forcer avec le bouton ↺ Sync.
💡 Codes absences : A = Absent C = Congé T = Terrain V = Vacances · Vide = Présent
⚡ CONNEXION GOOGLE SHEETS
Colle les IDs de tes Sheets — l'ID est dans l'URL : docs.google.com/spreadsheets/d/[CET ID]/edit
⏳ En attente — clique sur "Connecter" ou "Démo"
